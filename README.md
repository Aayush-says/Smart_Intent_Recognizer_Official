# Smart Intent Recognizer

AI-powered real-time intent recognition system using BERT, TensorFlow, and Gradio.

## Features
- Real-time intent prediction
- Fine-tuned BERT model
- Interactive Gradio UI
- SNIPS dataset integration
- End-to-end NLP pipeline

## Tech Stack
Python, TensorFlow, Hugging Face Transformers, BERT, Gradio

## Installation
pip install -r requirements.txt

## Train Model
python train_model.py

## Run Application
python app.py
